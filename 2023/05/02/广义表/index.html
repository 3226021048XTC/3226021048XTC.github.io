广义表

广义表是线性表的推广，也称为列表（lists）；

约定：

  • 第一个元素是表头(Head)，其余元素组成的表是表尾(Tail)；

  • 用小写字母表示原子类型，用大写字母表示列表；

广义表中元素既可以是原子类型，也可以是列表(子表)； 当每个元素都为原子且类型相同时，就是线性表；

特点：

  • 有次序性：一个直接前驱和一个直接后继；

  • 有长度：表中元素个数；

  • 有深度：表中括号的重数；

  • 可递归：自己可以作为自己的子表；

  • 可共享：可以为其他广义表所共享（共享时可以不必列出子表的值，通过子表的名称引用）；

由于广义表中的数据元素可以具有不同的结构，难以用顺序存储结构表示通常采用链式存储结构，每个数据用一个节点表示；

用原子节点和表节点表示两种不同的数据结构；

表节点由标志域，指示表头的指针域和指示表尾的指针域组成；

原子节点由标志域和值域组成；

//广义表的头尾链表存储表示
typedef enum {ATOM,LIST}ElemTag;//ATOM==0,原子,LIST==1;子表
typedef struct GLNode{
    ElemTag tag;//公共部分，用于区分原子结点和表结点
    union {
    AtomType atom;//原子结点的值域
    struct {struct GLNode *hp,*tp;}*ptr;//ptr为表结点的指针域，hp和tp分别
                                        //指向表头和表尾
}；
}*GList;  //广义表类型的GList为一种扩展的线性链表

//广义表的扩展线性链表存储表示
typedef enum {ATOM,LIST};//ATOM==0,原子,LIST==1;子表
typedef struct GLNode{
    ElemTag tag;//公共部分，用于区分原子结点和表结点
    union {
    AtomType atom;//原子结点的值域
    struct GLNode *hp;//表结点的表头指针
}；
    struct GLNode *tp;//相当于线性链表的next，指向下一个元素结点；
}*GList;  //广义表类型的GList为一种扩展的线性链表
